## Getting Started

Install the dependencies

```sh
npm i
```

Run the application in development environment

```sh
npm run watch
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Test

Launches the test runner.

```sh
npm test
```

Generate coverage

```sh
npm run coverage
```
